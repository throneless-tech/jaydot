# Jaydot

[![Built with Astro](https://astro.badg.es/v2/built-with-astro/small.svg)](https://astro.build)

A website for Jaydot, a mission-based social enterprise focused on creating affordable housing solutions for people in the Washington DC area, particularly those who have experienced homelessness. 

Built with Astro, Payload CMS, and deployed on Netlify.

## Getting started

Clone this theme locally and run any of the following commands in your terminal:

| Command           | Action                                       |
| :---------------- | :------------------------------------------- |
| `npm install`     | Installs dependencies                        |
| `npm run dev`     | Starts local dev server at `localhost:4321`  |
| `npm run build`   | Build your production site to `./dist/`      |
| `npm run preview` | Preview your build locally, before deploying |
